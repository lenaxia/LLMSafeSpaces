LLMSafeSpaces Helm Charts

This is the Helm chart repository for LLMSafeSpaces. There's nothing to see here in a browser — use helm to consume it.

Install

helm repo add llmsafespaces https://lenaxia.github.io/LLMSafeSpaces/
helm repo update
helm search repo llmsafespaces
helm install llmsafespaces llmsafespaces/llmsafespaces --version 0.3.0
